# Camper Api

> Backend Api for the Camper application, which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

### Install Dependecies

```
npm install
```

## Run app

```
# Run in dev mode
npm run dev

# Run prod mode
npm start
```
